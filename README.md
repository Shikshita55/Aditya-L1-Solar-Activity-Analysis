# Astronomy & Astrophysics: Heliophysics with Aditya-L1 SUIT Data

## Overview

This repository contains the analysis performed during the India Space Academy Summer Training Program 2026, focusing on heliophysics using observations from India's first solar mission, Aditya-L1.

The project investigates solar atmospheric structures and active regions using SUIT (Solar Ultraviolet Imaging Telescope) data through scientific image processing, segmentation techniques, temporal evolution analysis, and intensity-based energy diagnostics.

---

## Objectives

- Process Aditya-L1 SUIT solar ultraviolet observations
- Detect and analyze solar active regions
- Study temporal evolution of chromospheric structures
- Perform intensity-based magnetic flux correlation analysis
- Compare image segmentation techniques
- Develop scientific visualizations of solar activity

---

## Dataset

Mission:
Aditya-L1 Solar Mission

Instrument:
Solar Ultraviolet Imaging Telescope (SUIT)

Observation:
NB03 Mg II wavelength channel

Data Type:
FITS solar images

Observation Period:
18 February 2024 – 25 February 2024
(Independent exploration dataset)

---

## Methodology

The pipeline includes:

1. Data extraction from FITS files
2. Solar image preprocessing
   - Normalization
   - Gaussian filtering
   - Noise reduction

3. Active Region Detection
   - Otsu thresholding
   - Adaptive thresholding
   - K-means clustering

4. Temporal Evolution Analysis

5. Intensity and magnetic flux proxy correlation

6. Advanced Visualization
   - 2D solar maps
   - 3D solar surface representation
   - Solar activity evolution animation
